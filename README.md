# 📁 Local File Share

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/yourusername/local-file-share)
[![Node.js](https://img.shields.io/badge/Node.js-14.x+-green.svg)](https://nodejs.org/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Обмен файлами по локальной сети между любыми устройствами через браузер.

## ✨ Возможности

- 📤 Загрузка файлов через браузер
- 📥 Скачивание отдельных файлов
- 📦 Скачивание всех файлов одним архивом
- 🔤 Поддержка кириллицы в именах файлов
- 📱 Работает на любых устройствах
- 🚀 Автоматический запуск при старте системы

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск сервера
npm start

# Открыть в браузере
http://localhost:3000
